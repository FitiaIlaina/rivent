# RI’vent – Plateforme de réservation de billets d’événements en ligne
## Participants: 2

**RI’vent** est une plateforme web qui permet aux utilisateurs de **réserver des billets pour des événements en ligne ou en présentiel** de manière simple et rapide.  
Elle vise à offrir une expérience fluide pour **consulter, sélectionner et sécuriser des billets**, tout en permettant aux organisateurs de **gérer facilement leurs événements**.
Ce projet est développé avec **Flask**, utilisant **HTML/CSS** et **Bootstrap** pour une interface responsive et moderne.

## Objectifs du projet

- Permettre aux utilisateurs de **rechercher et réserver des billets** pour différents événements
- Fournir un **tableau de bord simple** pour les organisateurs d’événements
- Gérer les informations liées aux **billets, événements et utilisateurs**
- Offrir une **expérience utilisateur responsive et intuitive**


## Technologies utilisées

- **Flask** – Framework backend Python pour gérer les routes, la logique métier et les bases de données
- **HTML / CSS** – Structure et styles de base pour les pages web
- **Bootstrap** – Design moderne et responsive, composants UI
- **MySQL** (au choix) – Base de données pour stocker utilisateurs, événements et billets
